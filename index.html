Welcome to Cucumber/C++

Cucumber/C++ is an editor of Cucumber/Gherkin, a language of specification by example in the world of BDD. It could generate C++ skeleton implementation from Gherkin.

Prerequisite

  1. Qt5.6
  2. C++ compiler supporting C++14, for example Visual Studio 2015, gcc5.2.
  3. Bison

Features

  1. Keywords highlight
  2. Pretty formatting
  3. C++ skeleton of implementation generating
